# Домашнє завдання: Linux адміністрування та автоматизація 

Цей репозиторій містить практичне завдання з теми **«Linux адміністрування»**. Мета роботи — автоматизувати встановлення базового стеку інструментів DevOps-інженера за допомогою Bash-скрипту.

##  Опис завдання
Створити Bash-скрипт (`install_dev_tools.sh`), який автоматично розгортає Docker, Docker Compose, Python (3.9+) та Django на системах Ubuntu/Debian. 

##  Інструкція із запуску

1. **Перейдіть у потрібну гілку проєкту:**
   ```bash
   git checkout -b lesson-3
   ```
   Скрипт має властивість ідемпотентності — перед встановленням кожного інструменту виконується перевірка, чи не встановлений він уже в системі.


##  Структура проєкту
* `install_dev_tools.sh` — головний скрипт автоматизації.
* `README.md` — документація до завдання.

##  Вимоги до системи
* **ОС:** Ubuntu / Debian
* **Права:** Доступ до `sudo` для встановлення пакетів.

##  Зробіть скрипт виконуваним:
Bash
 ```bash
chmod u+x install_dev_tools.sh
```

## Запустіть скрипт:
Bash
```bash
./install_dev_tools.sh
```
